# Backend proyecto MeteoEscuela 25/26


## Instalación

#### Clonar el repositorio

```
git clone https://github.com/eafernandezm01/open-meteo-backend.git
cd open-meteo-backend
```

#### Instalación de dependencias

```zsh
npm install
```

## Arranque del servicio

```zsh
npm start
```
